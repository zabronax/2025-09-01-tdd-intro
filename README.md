## Commands

- Run all tests once:

    ```sh
    dotnet test
    ```

- Run tests for a single project:

    ```sh
    dotnet test <PROJECT_PATH>
    ```

## Test Packages

- Testing Web APIs:

    ```sh
    dotnet add <PROJECT_PATH> package Microsoft.AspNetCore.Mvc.Testing
    ```
